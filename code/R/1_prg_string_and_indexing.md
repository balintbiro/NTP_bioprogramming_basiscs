``` r
#the return value of the typeof() function is gonna be the actual data type of a variable
typeof(message)
```

    ## [1] "character"

``` r
#purely typed number have a datatype "double" which refers to the double precision floating point number
number <- 123
typeof(number)
```

    ## [1] "double"
